# :open_file_folder: ReactRobots

Minimal React App

:link: [View Demo](https://react-robots-3cpq375oz-marius-paduraru.vercel.app/)

## Installation

Clone this repo and npm install.

```bash
npm i
```

## Usage

### Development server

```bash
npm start
```

You can view the development server at `localhost:4200`.
